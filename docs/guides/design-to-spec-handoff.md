# Design to Spec Handoff

## Purpose

Turn a design, PRD, or product request into one narrow implementation-ready task.

## Handoff rule

The planner should produce:
- objective
- scope
- non-goals
- validation plan
- likely target files

The executor should not need to rediscover the task boundary from scratch.

## Rule

If the design implies multiple separate slices, create one spec for the smallest coherent slice first.
